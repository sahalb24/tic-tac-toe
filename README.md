# Tic-Tac-Toe Game

## Overview

This project is a simple command-line Tic-Tac-Toe game developed using Python. Two players can take turns placing their marks (X and O) on a 3×3 board. The game automatically detects the winner or a draw.

## Features

- Two-player gameplay
- Displays a 3×3 game board
- Checks for winning combinations
- Detects draw condition
- Simple command-line interface

## Technologies Used

- Python 3

## How to Run

```bash
python tic_tac_toe.py
```
